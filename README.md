# 📘 Optimization Chapter 7 Direct Methods Interactive Webpage

An interactive bilingual study webpage for **Chapter 7: Direct Methods** from *Algorithms for Optimization*.  
This project presents the core ideas of **derivative-free optimization** in a clear, visual, and beginner-friendly way, with structured method transitions, interactive illustrations, and quiz-based review.

---

## ✨ Project Highlights

- 🎯 **Focused on Chapter 7: Direct Methods**
- 🌐 **Interactive HTML webpage**, deployable with GitHub Pages
- 🧠 **Bilingual explanations (English + 中文)** for key concepts
- 🔗 **Method-to-method logic chain** showing:
  - what each method does,
  - its limitation,
  - and why the next method is introduced
- 🎨 **Vintage paper / ink-inspired interface design**
- 📊 **Visual and dynamic interactions** for algorithm intuition
- 📝 **Quiz module** with scoring, translation, and explanations
- 📚 **Summary section** for revision and quick review

---

## 📖 Covered Algorithms

This webpage includes six classical derivative-free optimization methods from Chapter 7:

1. **Cyclic Coordinate Search**
2. **Powell’s Method**
3. **Hooke–Jeeves Method**
4. **Generalized Pattern Search**
5. **Nelder–Mead Simplex Method**
6. **DIRECT**

---

## 🧩 What This Project Helps You Understand

This webpage is designed to help students understand:

- why **derivative-free methods** are needed
- how each **direct method** works
- how one method’s **limitation motivates the next**
- the difference between **local** and **global** direct methods
- how to compare methods in terms of:
  - search directions
  - cost of function evaluations
  - convergence intuition
  - best-use scenarios

## 🚀 Deployment

This project can be deployed directly using **GitHub Pages**.

### Steps
1. Upload the final HTML file
2. Rename it to `index.html`
3. Push it to the root of your GitHub repository
4. Enable **GitHub Pages** in repository settings
5. Select:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main`
   - **Folder**: `/ (root)`
